SeleniumTemplates
=================

Step 1:Run 'bundle' in the SeleniumTemplates directory
Step 2:Add server urls and your Sauce key to spec_helper.rb in: spec/QA/
                                                                spec/QA/ipad
                                                                spec/QA/iphone4
                                                                spec/QA/iphone5
                                                                spec/QA/iphone6
                                                                spec/QA/android
step 3:Run tests from script/QA/shorttests.rb or script/QA/iphone6.rb
